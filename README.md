# dotfiles

  Dotfile allow you to customize your system. Here I provide mine.

## Installation

    $ git clone https://github.com/abhikandoi2000/dotfiles.git ~/.dotfiles
    $ cd ~/.dotfiles
    $ ./bootstrap

This will symlink appropriate dotfiles in `.dotfiles` to your home directory.
So no hassles.
